## Installation Guide

### General Installation Steps

1. **Choose a template** that fits your needs
2. **Navigate** to the template directory
3. **Read** the template-specific README.md for detailed instructions
4. **Open** `index.html` in a browser or set up a local server
5. **Customize** according to the template's documentation

### Local Development Server Options

All templates can be run with a simple local server:

**Python 3:**
```bash
python -m http.server 8000
```

**Node.js (http-server):**
```bash
npx http-server -p 8000
```

**PHP:**
```bash
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

### Special Requirements

- **Portfolio 2, 4, 5**: Require PHP server for contact form functionality
- **Portfolio 6**: Optional Gulp build system (Node.js required for development mode)

## File Organization

### Standard Structure
```
template-name/
├── index.html          # Main page
├── README.md          # Documentation
├── assets/ or CSS/    # Stylesheets
├── JS/ or assets/js/  # JavaScript
├── Images/ or assets/img/ # Images
└── vendor/ or assets/vendor/ # Third-party libraries
```

### Special Cases
- **Portfolio 1**: Separate CSS/, JS/, Images/, Fonts/ folders
- **Portfolio 2, 5**: assets/ folder with subfolders
- **Portfolio 3**: css/, js/, images/, fonts/ folders (lowercase) + scss/ for source files
- **Portfolio 4**: css/, js/, images/ folders (lowercase)
- **Portfolio 6**: public_html/ folder for main files
- **Portfolio 7**: css/, js/, images/, font/, sass/ folders

## Tips for Choosing a Template

1. **Multi-page vs Single-page**: 
   - Choose Portfolio 1 for multi-page sites
   - Choose Portfolio 2, 3, 4, 5, 6, 7 for single-page sites

2. **Navigation Style**:
   - Side navigation: Portfolio 1
   - Sidebar: Portfolio 5
   - Top navigation: Portfolio 2, 3, 4, 6, 7

3. **Build System**:
   - Need Gulp: Portfolio 6
   - SCSS/SASS source files: Portfolio 3, 7
   - Simple HTML/CSS: Portfolio 1, 2, 4, 5

4. **Contact Form**:
   - Need PHP form: Portfolio 2, 5
   - Has contact form: Portfolio 3, 4, 7
   - No form: Portfolio 1, 6

5. **Color Customization**:
   - Pre-built schemes: Portfolio 1
   - SCSS/SASS for easy customization: Portfolio 3, 7
   - Dark/Light mode toggle: Portfolio 7
   - Full customization: All templates
